## ⚙️ Como usar este repositório

```bash
# Clone o repositório
git clone https://github.com/pabl0maciel/Programas-e-Projetos.git

# Acesse a pasta do projeto desejado
cd Programas-e-Projetos/Projetos/<nome-do-projeto>

# Siga o README.md de cada projeto para instruções específicas
```

A maioria dos projetos Python requer apenas:
```bash
pip install -r requirements.txt  # se houver
python <nome_do_script>.py
```
