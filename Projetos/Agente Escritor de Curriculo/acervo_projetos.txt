PROJETO 1
Nome do projeto: Análise Exploratória de Dados
Objetivo principal: Realiza análise de dados de vendas a partir de um arquivo CSV, respondendo perguntas de negócio sobre desempenho de vendas com diferentes categorias e segmentações e gerando visualizações para demonstrar os resultados.

Tipo do projeto: EDA

Palavras-chave ATS relevantes: Python; Pandas; Numpy; Matplotlib; Seaborn; DateTime; análise exploratória de dados; CSV; visualização de dados; gráfico de barras; gráfico de pizza; gráfico de linha; segmentação de vendas; análise de vendas
Descrição adaptável: Realizei uma análise exploratória de dados de vendas em Python, utilizando Pandas, Numpy, Matplotlib e Seaborn para responder perguntas de negócio e gerar visualizações por data, estado, cidade, segmento, categoria e subcategoria.

Tipo do projeto: Automação
Nível de complexidade percebido: Intermediário — envolve automação de interface gráfica com pyautogui, leitura de dados via CSV e integração entre entrada de dados e ações no navegador, além de dependência de coordenadas de tela.
Palavras-chave ATS relevantes: Python; pyautogui; pandas; automação de processos; automação de tarefas; leitura de CSV; preenchimento automático de formulários; automação de navegador; simulação de teclado e mouse
Descrição adaptável: Desenvolvi uma automação em Python utilizando pyautogui e pandas para cadastro de produtos em sistema web, integrando leitura de dados via CSV com preenchimento automático de formulários por meio de simulação de ações de teclado e mouse.

Descrição adaptável: Desenvolvi um sistema em Python para criação e população automatizada de banco de dados PostgreSQL, utilizando SQLAlchemy e Faker para gerar dados sintéticos realistas e estruturar entidades como clientes, produtos, pedidos, vendas e pagamentos com regras de negócio aplicadas.
